Pta Contest in School

Posted on 2019-03-16

Today we have a school contest named PTA Ladder Exam. After 3 hours program, I won the 1st place with 189 scores.

It’s easy at first, so I kept the 1st place with my program speed and Python skill. But when it comes to the last 3 problems, and things got tough. I failed in 3 test points in L2-2, 2 test points in L2-3. And now I went back to 2nd.

There are two choices at that time: debug these two programs, or challenge the last problem(30 scores totally). I choose the latter. After 20 minutes, I fully understand the meaning of the problem and start to code(now there are nearly 35 minutes). When I finish the code, there are 18 minutes left. But bugs appeals. What’s so bad is, my IDE is going to break out and I am not familiar with it. If I can’t get a score from this problem, I will only get 3rd finally. And now there are only 10 minutes, and I can’t run the code still. When I finally repair the IDE, there are only 8 minutes. I find the bugs rapidly, and finally, submit the code in the last 3 minutes. I get 24 from it and go back to the 1st place. Then the contest ends.

It’s really very exciting, which is merely the fun of program contest.
